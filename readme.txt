== Description ==


very simple plugin that converts a multi users (or anyones i guess) default two column dashboard layout to one column. good for displaying custom dashboard plugins that require a wide layout.

== Installation ==

1. Upload `1col-default.php` to the `/wp-content/mu-plugins/` directory
